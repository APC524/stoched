Detailed Description

Class MidpointLeap implements Realization step() function using the midpoint tau leap approximate algorithm of Gillepsie (2001). The method is analogous to the deterministic midpoint (2nd-order Runge-Kutta) method for the numerical solution of ordinary differential equations.
