20 /* This is the successor to xorshift128+. It is the fastest full-period
21  generator passing BigCrush without systematic failures, but due to the
22  relatively short period it is acceptable only for applications with a
23  mild amount of parallelism; otherwise, use a xorshift1024* generator.
24 
25  Beside passing BigCrush, this generator passes the PractRand test suite
26  up to (and included) 16TB, with the exception of binary rank tests,
27  which fail due to the lowest bit being an LFSR; all other bits pass all
28  tests. We suggest to use a sign test to extract a random Boolean value.
29 
30  Note that the generator uses a simulated rotate operation, which most C
31  compilers will turn into a single instruction. In Java, you can use
32  Long.rotateLeft(). In languages that do not make low-level rotation
33  instructions accessible xorshift128+ could be faster.
34 
35  The state must be seeded so that it is not everywhere zero. If you have
36  a 64-bit seed, we suggest to seed a splitmix64 generator and use its
37  output to fill s. */
